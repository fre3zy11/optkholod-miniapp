### 6. Все цены с НДС

Теперь цены в каталоге, карточке товара, корзине и заказе подписаны как цены с НДС.

Пример:

```text
155 ₽/кг · все цены с НДС
```

В сообщении заказа в Telegram тоже будет:

```text
Цена: 155 ₽/кг с НДС
Итого: 1 550 ₽ с НДС
```

### 7. Убраны доллары и евро

Переключатель валют USD/EUR убран из верхнего меню.

Остался только рубль:

```text
₽ с НДС
```

### 8. Исправлен ввод веса, где минимум 10 кг

Раньше нельзя было нормально стереть `10`, потому что поле сразу возвращало минимальное значение.

Теперь можно:

1. открыть карточку товара;
2. нажать на поле веса;
3. стереть `10`;
4. написать свое значение, например `25`, `50`, `100`.

Если оставить пусто или написать меньше минимальной упаковки, после выхода из поля вернется минимальный вес.

### 11. Добавлена админка для быстрой смены цены

Админка находится здесь:

```text
http://127.0.0.1:8000/admin
```

На сервере будет так:

```text
https://твой-домен.ru/admin
```

В админке можно:

- быстро менять цену за кг;
- менять вес упаковки;
- добавлять новый товар;
- удалять товар;
- видеть путь к фото товара.

Админка редактирует файл:

```text
web/products.json
```

Mini App автоматически загружает товары из этого файла.

### 12. Заказ приходит в твой Telegram

После нажатия на кнопку `Оформить заказ` Mini App отправляет заказ на сервер, а сервер отправляет сообщение в твой Telegram.

В сообщении будет:

- кто сделал заказ;
- Telegram ID покупателя;
- username, если есть;
- список товаров;
- вес товара;
- количество;
- цена за кг;
- сумма по товару;
- итоговая сумма;
- пометка про НДС;
- пометка про минимальный заказ от палета и самовывоз из Москвы.

Пример сообщения:

```text
🧾 Новый заказ ОптХолод

👤 Покупатель:
Иван @ivan
Telegram ID: 123456789

📦 Товары:
1. Картофель фри Global Fries 10 мм 10 кг
   Вес: 10 кг × 2 шт.
   Цена: 155 ₽/кг с НДС
   Сумма: 3 100 ₽

💰 Итого: 3 100 ₽ с НДС
ℹ️ Минимальный заказ от палета.
📍 Самовывоз со склада в Москве.
```

### 13. Сообщение после оформления заказа

После успешной отправки заказа покупатель увидит окно:

```text
Спасибо за заказ!
С вами свяжется наш менеджер
Для обсуждения деталей)
```

# Важный момент про GitHub Pages

Если загрузить только `web/` на GitHub Pages, то каталог будет работать, но:

- админка не сможет сохранять цены;
- заказ не сможет приходить в Telegram;
- Python-бот не будет работать постоянно.

Почему: GitHub Pages умеет показывать только статические файлы HTML/CSS/JS. Он не запускает Python и не может записывать файлы.

Для полной версии нужен сервер/VPS.

# Как запустить локально на компьютере

## 1. Распакуй архив

Распакуй проект в удобную папку, например:

```text
C:\projects\optxolodbot
```

## 2. Открой папку проекта в VS Code

Открой именно корневую папку, где лежит:

```text
run.py
web/
admin/
requirements.txt
.env.example
```

## 3. Создай виртуальное окружение

В терминале VS Code:

```bash
py -3 -m venv .venv
```

Активируй его:

```bash
.venv\Scripts\activate
```

## 4. Установи зависимости

```bash
pip install -r requirements.txt
```

## 5. Создай `.env`

Скопируй файл:

```text
.env.example
```

и переименуй копию в:

```text
.env
```

Заполни его:

```env
BOT_TOKEN=твой_токен_бота
WEBAPP_URL=https://твой-домен.ru/
ORDER_CHAT_ID=твой_telegram_id
ADMIN_PASSWORD=твой_пароль_для_админки
PORT=8000
```

## 6. Как узнать `ORDER_CHAT_ID`

Запусти бота:

```bash
python run.py
```

Открой бота в Telegram и напиши:

```text
/id
```

Бот пришлет твой Telegram ID. Его нужно вставить в `.env`:

```env
ORDER_CHAT_ID=123456789
```

После изменения `.env` перезапусти `python run.py`.

## 7. Запуск

```bash
python run.py
```

Mini App локально:

```text
http://127.0.0.1:8000/
```

Админка локально:

```text
http://127.0.0.1:8000/admin
```

# Как пользоваться админкой

1. Открой `/admin`.
2. Введи пароль из `.env`, поле `ADMIN_PASSWORD`.
3. Нажми `Войти`.
4. Найди товар через поиск.
5. Измени цену за кг.
6. Нажми `Сохранить`.
7. Обнови Mini App — новая цена подтянется из `web/products.json`.

Файл, где хранятся товары:

```text
web/products.json
```

# Как добавить новый товар

В админке есть блок `Добавить новый товар`.

Нужно заполнить:

- название RU;
- название EN, можно не заполнять;
- категория RU;
- цена за кг;
- упаковка в кг;
- путь к картинке;
- описание.

Пример пути к картинке:

```text
assets/products/fries-10mm.webp
```

Картинку нужно положить сюда:

```text
web/assets/products/
```

Например:

```text
web/assets/products/new-product.webp
```

А в админке указать:

```text
assets/products/new-product.webp
```

# Как заказ отправляется в Telegram

Mini App отправляет POST-запрос:

```text
/order
```

Сервер принимает заказ и отправляет сообщение в Telegram ID из `.env`:

```env
ORDER_CHAT_ID=123456789
```

Если `ORDER_CHAT_ID` не указан, заказ не сможет прийти тебе в Telegram.

# Что загружать на сервер

Для полной версии надо загружать весь проект:

```text
run.py
.env
requirements.txt
web/
admin/
```

Не загружай публично:

```text
.env
.venv/
.git/
```

`.env` должен быть только на сервере.
